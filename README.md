# Dopamine Gacha Zombie Defense

A fast-paced pixel-art gacha + zombie defense game built with React, TypeScript and Vite.

## Local Development

Prerequisites: Node.js 20+


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
3. Build production bundle:
   `npm run build`
4. Preview production build:
   `npm run preview`

## Tech Stack
- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- Motion + Lucide React
